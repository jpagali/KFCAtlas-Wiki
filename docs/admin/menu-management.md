# Menu Management

The Atlas Platform gives franchisees and regional teams the ability to manage their menu directly — activating and deactivating items, updating pricing, and controlling which items are available on which channels.

## How to Access

**Admin Portal → Restaurants → [Your Restaurant] → Menu**

## What You Can Manage

| Action | How |
|---|---|
| Activate / deactivate an item | Menu → Items → Toggle on/off |
| Update item price | Menu → Items → [Item] → Edit Price |
| Set channel availability | Menu → Items → [Item] → Channel Availability |
| Create a temporary 86 (out of stock) | Menu → Items → Mark as Unavailable |
| Update item description | Menu → Items → [Item] → Edit (regional admin only) |

## Channel-Specific Menus

Items can be made available on specific channels only. For example:
- A **Family Feast** bundle available for Dine-in and Takeaway only — not Delivery
- A **Delivery Exclusive** combo available only through the delivery channel

To configure this: **Menu → Items → [Item] → Channel Availability → Select channels**

:::info
Menu changes apply to your restaurant only. They do not affect any other location, even within the same franchise group.
:::

## Market Pricing Notes

### Japan 🇯🇵
Prices are set in JPY. Tax (10% consumption tax) is included in displayed prices by default — this is a legal requirement. Do not set pre-tax prices.

### Australia 🇦🇺
Prices are set in AUD. GST (10%) is included in displayed prices. Delivery prices may differ from in-store prices — this is configurable per item under Channel Availability → Delivery Pricing.

## FAQs

**Can I change the name of a menu item?**
Item names and descriptions are managed at the regional level to ensure brand consistency. Contact your BMU if a name needs updating.

**How quickly do menu changes apply?**
Activations and deactivations apply within 5 minutes. Price changes apply within 15 minutes.

**What happens if I deactivate an item that's in an active combo?**
The platform will warn you. You can proceed, but the combo will automatically become unavailable if a required item is deactivated. Reactivating the item will restore the combo.

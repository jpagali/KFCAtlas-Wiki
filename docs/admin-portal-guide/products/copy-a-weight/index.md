# Copy a Weight

## What this guide covers

Duplicates a weight entry to reduce setup time for similar configurations.

## Steps

**Step 1:** Start by going to the Products screen by clicking here.

**Step 2:** Click the Weights tab.
**Step 3:** You can search Option Values by entering the Name or Code.

**Step 4:** Click the 3 dots to reveal a panel. Click Copy.

![Step 2](./img/step-2.png)

**Step 4:** Fill in each “*”required field and other valuable information.

**Step 5:** When you are finished adding in all the information, click the Create Weight button.

![Step 3](./img/step-3.png)

## Notes

:::note
There are other options in the window  but for this step we are just looking at Copy. Others are discussed else where. Please go to the Table of Contents to find where.
:::

:::note
If you need to stop your creation click here. Please be aware that your info will not be saved.
:::

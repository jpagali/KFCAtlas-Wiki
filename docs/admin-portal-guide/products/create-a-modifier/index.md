# Create a Modifier

## What this guide covers

Creates an add-on or upsell item (e.g. 'Extra Cheese') that can be linked to products, allowing operators to offer customisations that may affect pricing.

## Steps

**Step 1:** Start by going to the Products screen by clicking here.

![Step 1](./img/step-1.png)

**Step 2:** Click the Modifiers tab.

![Step 2](./img/step-2.png)

**Step 3:** Click the “+ Create New Modifier” button.

**Step 4:** Fill in each “*”required field and other valuable information.

**Step 5:** When you are finished adding in all the information, click the Create Modifier button.

![Step 3](./img/step-3.png)

## Notes

:::note

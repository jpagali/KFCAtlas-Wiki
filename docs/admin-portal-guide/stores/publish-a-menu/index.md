# Publish a Menu

## What this guide covers

Publishes the assigned menu to the store's live ordering channels from within the Stores screen.

## Steps

**Step 1:** Start by going to the Stores screen by clicking here.
**Step 2:** Select Publish Menu

![Step 2](./img/step-2.png)

**Step 3:** Select Stores

![Step 3](./img/step-3.png)

**Step 4:** Select the channels you would like to publish the menu to

![Step 4](./img/step-4.png)

**Step 5:** Select Publish

![Step 5](./img/step-5.png)

## Additional information

- Stores - Publish a Menu

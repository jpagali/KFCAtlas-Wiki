# View/Unassign a Store's Store Groups

## What this guide covers

Displays the store groups a store belongs to and allows operators to unassign the store from a group when needed.

## Steps

**Step 1:** Start by going to the Stores screen by clicking here.
**Step 2:** You can search stores by entering the Name, Number, or Franchise Code.

![Step 2](./img/step-2.png)

**Step 3:** Once you find the store you are looking for, click on the stacked dots to open the option window.

**Step 4:** Click on Store Groups.

![Step 3](./img/step-3.png)

## Notes

:::note
There are other options in the window  but for this step we are just looking at Store Groups. Others are discussed else where. Please go to the Table of Contents to find where.
:::

## Additional information

- Stores - Store Groups
- On this page you can view all of the store groups the store is assigned to.
- You can search/filter by store group name
- Clicking this button will open up this drawer that allow you to unassign the store from a store group

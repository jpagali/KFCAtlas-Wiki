# Add an Image to a Bundle

## What this guide covers

Assigns a display image to a bundle so customers see the correct combo visual when ordering.

## Steps

**Step 1:** Start by going to the Bundles screen by clicking here.

![Step 1](./img/step-1.png)

**Step 2:** Click this  button in the same row your bundle is in and then hit Images

![Step 2](./img/step-2.png)

**Step 2:** Fill out the content in this drawer. Image URL is what adds the image. Image Label is for labeling the image and alt text is for a description of what the image is

**Step 3:** Click save to save image to your bundle.

![Step 3](./img/step-3.png)

# Edit a Menu

## What this guide covers

Updates an existing menu's configuration, such as its name, assigned items, or channel settings.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.
**Step 2:** Click this button in the same row the menu you’re looking for is in and then hit Edit

![Step 2](./img/step-2.png)

**Step 3:** After making edits hit save

![Step 3](./img/step-3.png)

![Step 4](./img/step-4.png)

![Step 5](./img/step-5.png)

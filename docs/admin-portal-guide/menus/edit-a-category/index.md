# Edit a Category

## What this guide covers

Updates a category's name or configuration within a menu.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.
**Step 2:** Click on the categories folder

![Step 2](./img/step-2.png)

**Step 3:** Click this  button in the same row the category you’re looking for is in and then hit Edit

![Step 3](./img/step-3.png)

**Step 4:** Edit the information you want to and hit save afterwards.

![Step 4](./img/step-4.png)

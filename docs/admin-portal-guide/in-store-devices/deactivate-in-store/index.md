# Deactivate In-Store

## What this guide covers

Deactivates an in-store device in Atlas, preventing it from receiving menu updates or processing orders until reactivated.

## Steps

**Step 1:** Start by going to the In-Store Devices screen by clicking here.
**Step 2:** Click this button in the same row the device you want to configure is in and then hit Deactivate

![Step 2](./img/step-2.png)

**Step 3:** Click to deactivate the in store device

![Step 3](./img/step-3.png)

# Assign Promotions to Store Groups

## What this guide covers

Links existing promotions to store groups, making them active for all stores within those groups simultaneously.

## Steps

**Step 1:** Start by going to the Promotions screen by clicking here.
**Step 2:** Click this button to assign a promotion to store groups

![Step 2](./img/step-2.png)

**Step 3:** Select the promotions that you want to assign to store groups

![Step 3](./img/step-3.png)

**Step 4:** To continue to the next step click Next or the appropriate step above.

**Step 5:** Select the store groups that you want to have promotions applied to

**Step 6:** Click assign after selecting store groups

![Step 4](./img/step-4.png)

## Additional information

- Promotions - Assign Promotions to Store Group(s) (Also can do in Store Groups)
- Assign to store groups
- This is the Promotions screen where you  will see a list of all the promotions you have created, create new promotions, search for any you have created, edit and copy, add extra info in the Meta link and  assign them to Store Groups.  Promotions can only assigned to a Store Group and not a singular store.
- If you need to stop your creation click here but please be aware that your info will not be saved.

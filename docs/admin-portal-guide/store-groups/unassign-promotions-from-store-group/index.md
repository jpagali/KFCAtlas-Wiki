# Unassign Promotions from Store Group

## What this guide covers

Removes one or more promotions from a store group, deactivating them across all member stores.

## Steps

**Step 1:** Start by going to the Store groups screen by clicking here.
**Step 2:** To see all promotions assigned to a certain store group click this button and the click Promotions.

![Step 2](./img/step-2.png)

**Step 3:** To unassign a promotions from a store group, click this unassign button.

![Step 3](./img/step-3.png)

**Step 4:** After clicking Unassign on a promotion, a modal will pop up to make sure this was your intent. If so Just click Unassign Promotion.

![Step 4](./img/step-4.png)

## Additional information

- Store Groups - Unassign Promotions from Store Group
- On this screen you will see all the  store groups created. You can search for them and create a new group from this screen. As well as Edit, Delete and see the promotions assigned to that store group.

# Release 2026.03.19

Released on **March 19, 2026**.

This release fixes light-mode styling inconsistencies in the Atlas Peek prototype, resolves duplicate sidebar category names in the Admin Portal Guide, enhances navbar responsiveness for zoom levels, removes the Contribute button from the desktop navbar, and updates the Byte Capabilities credits page.

## Highlights

- **Atlas Peek prototype now renders correctly in light mode**
  The PICKUP/ASAP chips, topbar, navigation, and journey-shell bar now display with proper light-mode colors instead of defaulting to dark-mode styling.

- **Duplicate sidebar category names resolved**
  The Admin Portal Guide sidebar had nested categories with identical labels (Stores, Menus, Products, etc.). These have been renamed to "Tasks" subcategories to prevent translation conflicts and improve clarity.

- **Navbar now responds to browser zoom levels**
  The search bar in the navbar now shrinks progressively as the viewport narrows (from 280px down to hiding completely), ensuring navigation links remain visible at various zoom levels.

- **Contribute button removed from desktop navbar**
  The Contribute button was removed from the desktop navbar to reduce clutter. Users can still access the contribution studio via the footer CTA.

- **Search placeholder updated**
  The navbar search bar now uses "What's on your mind?" as the placeholder text for a more conversational feel.

- **Hero search bar width adjusted**
  The hero search bar max-width was reduced from 720px to 600px for better visual balance.

- **Byte Capabilities credits updated**
  The credits page now reflects "The Byte Capabilities" and includes a link to the original Confluence documentation.

## What Changed

### Atlas Peek Prototype

- Added light-mode styles for `.journey-shell-chip` (PICKUP/ASAP chips) with black text and transparent background
- Added light-mode background for `.journey-shell-bar` (white background)
- Added light-mode styles for `.topbar` and `.web-nav` (white background)
- Added light-mode styles for `.pill` buttons in action sheets

### Sidebar Navigation

- Renamed duplicate "Stores" nested category to "Store Tasks"
- Renamed duplicate "Menus" nested category to "Menu Tasks"
- Renamed duplicate "Products" nested category to "Product Tasks"
- Renamed duplicate "Promotions" nested category to "Promotion Tasks"
- Renamed duplicate "Bundles" nested category to "Bundle Tasks"
- Renamed duplicate "Store Groups" (under Promotions) nested category to "Store Group Tasks"

### Navbar

- Added responsive breakpoints to shrink the navbar search bar based on viewport width:
  - 1500px+: Hide theme toggle
  - 1400px+: Hide contribute button
  - 1300px: Shrink search to 220px
  - 1200px: Shrink search to 180px
  - 1100px: Shrink search to 140px
  - 1000px: Hide search completely
- Removed contribute button from `docusaurus.config.js` navbar items
- Reduced navbar right items gap from 0.75rem to 0.5rem
- Increased navbar search bar max-width from 220px to 280px

### Search

- Updated navbar search placeholder from "Search articles and headings" to "What's on your mind?"
- Updated Japanese search placeholder to "気になることは？"
- Changed hero search bar max-width from 720px to 600px

### Documentation

- Updated Byte Capabilities credits page title and description
- Added Confluence link to Varima Henry and Wesley Weidemann attribution

## In Plain English

This release cleans up several loose ends. The Atlas Peek prototype now looks right in light mode, the sidebar navigation is clearer without duplicate labels, and the desktop navbar now adapts more gracefully when users zoom in or view on smaller screens.

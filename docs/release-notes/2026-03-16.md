# Release 2026.03.16

Released on **March 16, 2026**.

This release refreshes the Atlas Knowledge Center homepage with a more cinematic KFC-branded hero and improves the Admin Portal navigation structure so people can move through the guide with less friction.

## Highlights

- **The homepage hero now uses cinematic KFC footage**
  The Knowledge Center landing page now features a full-bleed motion hero using KFC cooking footage behind the headline, search, and primary calls to action.

- **The homepage feels more like a branded product surface**
  The hero treatment now carries more visual energy while keeping the main search and onboarding entry points front and center.

- **Admin Portal navigation is organized more clearly**
  The Admin Portal sidebar structure was cleaned up so major sections read more intentionally and support a more coherent breadcrumb and browsing experience.

- **The updated homepage still supports mobile layouts**
  The hero overlay and content layering were tuned so the text, search, and CTA controls remain readable on smaller screens.

## What Changed

### Homepage Experience

- Added a cinematic full-bleed video hero to the homepage
- Kept the hero headline, search, and CTAs layered above the video
- Added KFC-branded overlay treatment to protect readability over moving footage
- Preserved the existing onboarding entry points for Front-end Guide and Admin Portal Guide

### Navigation and Information Architecture

- Reworked the Admin Portal sidebar grouping to better reflect the main guide structure
- Promoted clearer top-level section entry points across Admin Portal content
- Improved how nested sections are surfaced for browsing and breadcrumb context

### Responsive Behavior

- Adjusted hero layering so background video remains behind interactive controls
- Tuned smaller-screen overlay behavior to keep the homepage content legible on mobile

## In Plain English

This release makes the Knowledge Center feel more unmistakably KFC when you land on it, while also making the Admin Portal documentation easier to move through once you start exploring.

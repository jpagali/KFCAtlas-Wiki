This standalone preview is intentionally fake in the same way a brainstorm should be: it mirrors the real layout and content hierarchy closely, but the point is the color token strategy. The production takeaway is that the Knowledge Center can keep its current cinematic and editorial structure while introducing a light mode built from warm neutrals, restrained red accents, and lower-cost theme variables.
Preview intent: color-system exploration only, not production UI. Single-file HTML with inline CSS and JavaScript for fast discussion.

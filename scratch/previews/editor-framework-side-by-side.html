What is intentionally fake here

No repo wiring, no Docusaurus integration, no asset uploads, and no persistence. This is just a safe UX spike for comparison.

If this direction feels right, the next step is to replace the production editor surface with a real editor framework inside the contribute page while preserving the existing metadata, preview, and Markdown download flow.

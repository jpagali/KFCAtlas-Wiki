# リリース 2026.03.17

**2026 年 3 月 17 日**に公開されました。

このリリースでは、Atlas Wiki 全体に本格的な light/dark theme を導入するとともに、RNA Sneak Peek の注文プロトタイプも、より実アプリに近いレビュー体験へと磨き込みました。

## ハイライト

- **Atlas Wiki 全体で本格的な light/dark mode を利用可能に**
  ダークを既定にしつつ、ホーム、ドキュメント、`What's New`、Contribution Studio まで一貫して切り替わる light mode を導入しました。

- **テーマ切り替えを desktop / mobile の両方に配置**
  デスクトップのナビゲーションとモバイルのハンバーガーメニューの両方から theme を切り替えられるようにしました。

- **RNA Sneak Peek がより実アプリらしい注文体験に**
  staged slide のような見え方から離れ、配送先設定、店舗選択、メニュー、カート、チェックアウト、追跡までがつながるモバイル注文フローとして確認できるようにしました。

- **RNA prototype が light/dark と locale の切り替えに自然に対応**
  ページ外側では余計な言語コントロールを外し、prototype 自身の切り替えで確認できる形に整理しました。

- **メニュー体験と商品表現を改善**
  Deals & Rewards の配置を見直して検索を優先し、主要商品のビジュアルには実際のフード写真を使うようにしました。

- **Atlas Wiki の chrome と design-system docs を整理**
  サイト全体の名称を `Atlas Wiki` に寄せ、`RNA Sneak Peek` の専用 navbar CTA を追加し、Helium アーキテクチャ図も小さい画面で見やすくしました。

## 変更内容

### Theme Infrastructure

- Docusaurus の color-mode を再び有効にしつつ、ブランドに合わせた custom toggle UI を維持
- サイト全体の既定テーマを dark に設定
- 再利用可能な custom theme toggle component を追加
- デスクトップ navbar に theme toggle を追加
- モバイル hamburger menu の language toggle 下に theme toggle を追加
- theme の切り替えはページ再読み込み不要の client-side 挙動を維持

### RNA Sneak Peek Prototype

- RNA の埋め込み prototype を、より実アプリに近いモバイル注文フローへ再構成
- `Start My Order` を起点に、delivery / pickup を分岐させるセットアップ導線へ整理
- 価格表示やカート追加を有効にする前に確認できる browse-only menu 状態を追加
- 配送先入力、候補選択、配送先確認マップ、自動店舗割当を含む mock delivery flow を追加
- store や address を後から見直せる軽量な `Change` 導線を追加
- 注文後には、mock map、配達員ステータス、ETA、配送先カードを含む tracking surface を追加
- Atlas Wiki 内でもレビューしやすいよう、埋め込み iframe 用の immersive / fullscreen mode を追加
- 埋め込み prototype が、Atlas Wiki 側の light / dark theme 選択に追従するよう調整
- ページ外側の重複した language control を削除し、locale 切り替えを prototype 内へ一本化
- `/rna-sneak-peek` では Atlas Chat を非表示にし、prototype review に集中できるよう整理
- Dynamic Island 周辺の control 配置、fullscreen 動作、小さい画面向けの responsive 挙動を改善
- Deals & Rewards が検索を邪魔しないよう、menu hierarchy を見直し
- 主要 menu item のビジュアルを placeholder から実フード写真へ差し替え

### Global Visual System

- グローバル stylesheet の light / dark shared token を見直し
- 事実上 light-only に戻ってしまっていた prior dark override を削除
- navbar、sidebar、footer、search、共通 chrome が theme により自然に切り替わるよう調整

### Atlas Wiki Chrome

- メイン site chrome の名称を `Atlas Knowledge Center` から `Atlas Wiki` に変更
- デスクトップ navbar に `RNA Sneak Peek` の専用 CTA を追加
- 短いブランド名と CTA が折り返さず収まるよう、desktop navbar の間隔を調整
- mobile navbar では inline search、より分かりやすい sidebar header、Home shortcut、compact な theme toggle 表現を追加
- light mode の mobile hamburger menu で、Home / close CTA が見つけやすいよう配色コントラストを改善

### Design System Docs

- Helium design-system architecture diagram が tablet / mobile でも見やすくなるよう responsive 化
- 複数の inline style を再利用しやすい class に置き換え、保守しやすく整理
- foundations / market rows が小さい画面でも自然に折り返すよう改善

### Contribution Studio

- contribution / publishing studio を Atlas Wiki と同じ theme system に統合
- studio panel、editor surface、preview、toolbar、modal overlay を light / dark に追従するよう更新

### Documentation Readability

- dark mode における admonition の可読性を改善
- info、secondary note、tip / success、warning、danger の配色を見直し
- icon、link、heading、inline code、本文テキストのコントラストを dark mode でも保てるよう調整

## わかりやすく言うと

このリリースで、Atlas Wiki は本格的な theme 切り替えに対応し、RNA Sneak Peek もレビュー用の静的な見え方から、より本物の注文アプリらしい確認体験へ近づきました。

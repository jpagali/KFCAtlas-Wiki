# リリース 2026.03.16

**2026 年 3 月 16 日**に公開されました。

このリリースでは、Atlas Knowledge Center のホームページをよりシネマティックな KFC ブランド表現に刷新し、あわせて Admin Portal ガイドのナビゲーション構造も整理して、より辿りやすくしました。

## ハイライト

- **ホームページのヒーローに KFC の映像演出を追加**
  Knowledge Center のトップページに、見出し、検索、主要 CTA の背後で再生されるフルブリードの KFC 調理映像を追加しました。

- **トップページがよりブランドらしい第一印象に**
  ヒーローの見た目に動きと存在感を持たせつつ、主要な検索導線とオンボーディング導線は引き続き前面に保っています。

- **Admin Portal のナビゲーション構造を整理**
  サイドバー構造を見直し、主要セクションがより自然に理解できるようにすることで、パンくずや閲覧体験の一貫性を高めました。

- **モバイルでも読みやすさを維持**
  ヒーローのオーバーレイとレイヤー構成を調整し、小さい画面でもテキスト、検索、CTA が読みやすい状態を保てるようにしました。

## 変更内容

### ホームページ体験

- ホームページにシネマティックなフルブリード動画ヒーローを追加
- ヒーローの見出し、検索、CTA は動画の上に重ねて表示
- 動きのある映像上でも読みやすさを保つため、KFC ブランドのオーバーレイ表現を追加
- Front-end Guide と Admin Portal Guide への既存導線は維持

### ナビゲーションと情報設計

- Admin Portal サイドバーのグルーピングを再構成し、全体構造が把握しやすい形に調整
- Admin Portal 内の主要セクション入口をより分かりやすく整理
- ネストされたセクションの見せ方を改善し、閲覧やパンくず文脈を把握しやすく調整

### レスポンシブ対応

- 背景動画が操作要素の背面に留まるよう、ヒーローのレイヤー構成を調整
- モバイルでの可読性を保つため、小さい画面向けのオーバーレイ挙動を最適化

## わかりやすく言うと

このリリースで、Knowledge Center を開いた瞬間の印象がより KFC らしくなり、同時に Admin Portal ドキュメントも迷いにくく辿りやすくなりました。
